**Note:** It is now possible to stop build jobs, so the `deletecollection` permission for `jobs` must be added to build roles.

**Example: Build Role with `deletecollection` permission**
```yaml
apiVersion: rbac.authorization.k8s.io/v1
kind: Role
metadata:
  name: dial-mcp-manager
  namespace: mcp-build
rules:
  - apiGroups:
      - "batch"
    resources:
      - jobs
    verbs:
      - get
      - list
      - watch
      - deletecollection
      - create
      - delete
  - apiGroups:
      - ""
    resources:
      - pods
      - pods/log
    verbs:
      - get
      - list
      - watch
  - apiGroups:
      - ""
    resources:
      - secrets
      - configmaps
    verbs:
      - get
      - create
      - delete
```
